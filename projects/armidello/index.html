Designed for complete beginners, Armidello is an IDE for ARM assembly.

The project began as part of the work for Dr. Tom Clarke's functional programming course in F#. The aim was to create a replacement for VisUAL, the program currently used to teach ARM assembly in Dr. Clarke's computer architecture course.

Armidello is written entirely in F# but makes use of Fable, an F# to JavaScript transpiler, and the Electron framework.

{% include gallery caption="Armidello allows you to change the editor prefrences and save files so you can feel totally at home while writing all that assembly" %}

A number of features within Armidello have been developed specifically to assist beginners

  • The immediatly available memory map which intuitvely differentiates between contigious and non-contigious memory ensures users will never forget what's in memory.
  • The ability to switch number representations from binary, hexadecimal, and signed or unsigned decimal helps users who are new are new to the idea of number representations.

Armidello was developed by a team of three: Tom Carotti, Doug Brion, and myself. Tom worked mostly on the GUI, while Doug and I wrote the backend for the emulator.

The project is now being continued by Dr. Clarke with the aim of fully replacing VisUAL in time for the start of his course in October 2018. All the code for Armidello is available on GitHub here.
